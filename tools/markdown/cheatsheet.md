## Акценты ############################

| Код                       | Результат                   |
|---------------------------|-----------------------------|
| `*Курсив*`                | *Курсив*                    |
| `_Курсив_`                | _Курсив_                    |
| `**Полужирнный**`         | **Полужирнный**             |
| `__Полужирнный__`         | __Полужирнный__             |
| `***Полужирный курсив***` | ***Полужирный курсив***     |
| `___Полужирный курсив___` | ___Полужирный курсив___     |
| `~~Зачеркнутый~~`         | ~~Зачеркнутый~~             |
| `<u>Подчеркнутый</u>`     | <u>Подчеркнутый</u>         |
| `<sub>Подстрочный</sub>`  | Текст<sub>Подстрочный</sub> |
| `<sup>Надстрочный</sup>`  | Текст<sup>Надстрочный</sup> |
| `вн**утри**фразы`         | вн**утри**фразы             |
| `вн_утри_фразы`           | вн_утри_фразы (Не работает) |
